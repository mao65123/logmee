# プロダクト要件定義書

## 目的

フリーランスの時間管理・売上追跡を一元化するWebアプリケーション。
複数クライアントの稼働時間を正確に記録し、時給・固定報酬の売上を自動計算する。

## 解決する課題

1. **稼働時間の記録漏れ**: タイマー機能でリアルタイム計測。フローティングタイマーで画面遷移中も継続
2. **売上の把握が困難**: 時給×稼働時間の自動計算 + 固定報酬の月次管理で正確な売上把握
3. **レポート作成の手間**: 期間・クライアント指定で作業報告書を自動生成（HTML/PDF/CSV）
4. **複数クライアントの管理**: クライアントごとに時給・締め日・カテゴリ・案件を独立管理

## 機能要件

### 1. ダッシュボード

- 今月の稼働時間・推定売上・目標達成率の表示
- クライアント選択（横スクロールチップ + 検索パネル）
- タイマー開始/停止
- 作業説明・カテゴリ・案件・報酬タイプの入力
- タスクプリセットからのオートコンプリート
- 前回の作業を繰り返す機能
- 今日の作業一覧
- クライアント並び替え（ドラッグ&ドロップ）

### 2. 履歴（ログ）ページ

- 月別・日付別の作業履歴一覧
- クライアント・カテゴリ・案件によるフィルタリング
- フィルター結果の件数表示
- カテゴリ別・案件別の合計時間サマリー
- 各エントリの編集・削除
- 「月表示に戻す」「フィルター全解除」ボタン

### 3. レポートページ

- 日付範囲・クライアント・案件によるフィルタリング
- 3つの折りたたみセクション（対象データ・書類情報・表示オプション）
- デスクトップ: 左右分割レイアウト（設定 + ライブプレビュー）
- HTML/PDF/CSV出力
- レポートの保存・再利用

### 4. クライアント管理ページ

- クライアントの追加・編集・削除
- 名前・テーマカラー・時給・締め日の設定
- 作業カテゴリの管理
- タスクプリセットの管理
- 固定報酬は案件管理への誘導リンク

### 5. 案件管理ページ

- 案件の追加・編集・削除
- クライアント・ステータス（アクティブ/完了）によるフィルタリング
- 固定報酬額の設定
- 月次報酬ON/OFFバッジ表示（今月ON）
- デスクトップ: テーブル表示、モバイル: カード表示

### 6. 分析ページ

- サマリーカード（総時間・推定売上）
- 3タブ構成:
  - 概要: クライアント別円グラフ + サマリーテーブル
  - 日次: 日次棒グラフ
  - カテゴリ: カテゴリ別円グラフ + リスト
- 月別パフォーマンステーブル

### 7. 設定

- ユーザー名
- テーマカラー（単色 + グラデーション）
- 月間目標（稼働時間・売上）
- 通貨（JPY/USD）
- 通知設定
- ログアウト

### 8. フローティングタイマー

- タイマー実行中に全ページで表示
- ドラッグで位置変更可能
- 位置をlocalStorageに永続化
- タイマー情報（経過時間・クライアント名・作業説明）表示

### 9. 使い方ページ

- アプリの操作手順を説明

## 非機能要件

### パフォーマンス

- SPA（Single Page Application）で高速なページ遷移
- Vite による高速な開発サーバーとビルド
- useMemo によるグラフ計算の最適化

### セキュリティ

- Google OAuth による認証
- Supabase RLS（Row Level Security）で全テーブルにユーザー単位のアクセス制御
- APIキーはハードコード禁止（環境変数管理）

### データ永続性

- Supabase（PostgreSQL）にリアルタイム同期
- ローカルストレージにもバックアップ保存
- オフライン時はローカルストレージで動作

### レスポンシブデザイン

- モバイルファースト設計
- デスクトップ/タブレット/スマートフォン対応
- Safe Area対応（ノッチ付き端末）

### ブラウザ対応

- Chrome（推奨）
- Safari
- Firefox
- Edge
