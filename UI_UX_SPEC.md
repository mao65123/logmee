## デザインシステム

### カラーパレット

| 用途 | 値 | CSS変数 |
|---|---|---|
| テーマカラー | ユーザー設定可能（デフォルト: #FFD700） | `--theme-color` |
| テーマ背景 | テーマカラーまたはグラデーション | `--theme-bg-gradient` |
| コントラストテキスト | テーマに応じて自動計算 | `--theme-contrast-text` |
| 背景色 | #F8F9FA | body background |
| テキスト色 | #1E293B | body color |

### テーマカラーオプション

単色: 10色以上の選択肢（#FFD700, #3B82F6, #10B981 等）
グラデーション: 4パターン（夕焼け、オーロラ、サンセット、オーシャン）

### タイポグラフィ

- **フォント**: Noto Sans JP (Google Fonts)
- **ウェイト**: 300 (light), 400 (regular), 500 (medium), 700 (bold)
- **サイズ**: Tailwind デフォルト + カスタムサイズ（text-[9px], text-[10px], text-[11px] 等）

### 角丸

- カード: `rounded-2xl` (1rem) または `rounded-3xl` (1.5rem)
- ボタン: `rounded-2xl`
- チップ: `rounded-full`
- 入力: `rounded-xl`

### シャドウ

- カード: `shadow-lg shadow-black/5`
- ボタン: `shadow-xl shadow-black/20`（アクション系）
- フローティングタイマー: `shadow-2xl`

## コンポーネント設計

### 共通UIコンポーネント（UIComponents.tsx）

#### Button

```tsx
<Button
  variant="primary" | "secondary" | "danger" | "ghost"
  size="sm" | "md" | "lg"
  className="..."
  onClick={handler}
>
  テキスト
</Button>
```

- primary: テーマカラー背景 + 白テキスト
- secondary: グレー背景
- danger: 赤背景
- ghost: 透明背景

#### Card

```tsx
<Card className="..." onClick={handler}>
  コンテンツ
</Card>
```

白背景 + 角丸 + シャドウ

#### Input

```tsx
<Input
  label="ラベル"
  value={value}
  onChange={handler}
  type="text" | "number" | "date"
  placeholder="..."
/>
```

#### Select

```tsx
<Select
  label="ラベル"
  value={value}
  onChange={handler}
  options={[{ value: 'v', label: 'l' }]}
/>
```

## レスポンシブ対応

### ブレイクポイント

| サイズ | Tailwind | 用途 |
|---|---|---|
| < 640px | デフォルト | モバイル |
| >= 640px | `sm:` | - |
| >= 768px | `md:` | タブレット |
| >= 1024px | `lg:` | デスクトップ |

### モバイル固有対応

- ボトムナビゲーション: 固定表示
- Safe Area: `env(safe-area-inset-top/bottom)` でノッチ対応
- 案件ページ: テーブル → カード表示
- レポートページ: 左右分割 → 縦積み
- 数値入力: スピンボタン非表示
- スクロールバー: カスタム（6px幅、薄いグレー）

## モーダル・ダイアログ

- **設定モーダル**: スライドイン（右から or 下から）
- **編集モーダル**: 中央表示 + 背景オーバーレイ
- **確認ダイアログ**: ネイティブ `confirm()` 使用（削除確認等）
- **レポートプレビュー**: フルスクリーンモーダル（モバイル）/ サイドパネル（デスクトップ）

## アニメーション

- タイマー点滅: `animate-pulse`（記録中インジケーター）
- スピナー: `animate-spin-slow`（3秒周期）
- ページ遷移: なし（即時切替）
- ボタンホバー: `transition-colors`
